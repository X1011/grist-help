Type conversions#

When converting between different column types, Grist has sensible default behavior, but makes that behavior easy to revise. For example, suppose you have a column of integers. To convert that column to text, open the column options as described in Specifying a type, and find the column type section. Change the column type to text in the dropdown. You’ll notice that a “cancel/revise/apply” dialog opens beside the dropdown.
